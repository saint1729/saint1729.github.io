# Agentic Design Patterns

A hands-on guide to learning and implementing agentic design patterns based on the book *Agentic Design Patterns* by Antonio Gulli.

## About This Resource

This is a personal companion guide featuring:

- **Detailed Chapter Notes**: Comprehensive summaries and key insights from each chapter
- **Code Examples**: Practical implementations in LangChain, CrewAI, and Google ADK
- **Hands-on Learning**: Working examples you can run and modify

## Getting Started

Use the sidebar navigation to explore chapters sequentially. Each chapter includes:

- Conceptual explanation of the pattern
- Practical Applications & Use Cases
- Implementation examples in multiple frameworks that you can execute and experiment with

## Repository

All code and content is maintained in the [agentic-design-patterns](https://github.com/saint1729/agentic-design-patterns) GitHub repository.
